 Temperature Conversion Program in Python

A simple **menu-driven Python program** that converts temperature between:

- Celsius ↔ Fahrenheit  
- Celsius ↔ Kelvin  

This project is perfect for **beginners** who are learning:
- Python functions  
- Loops  
- Conditional statements  
- User input handling  


Features

✔️ Celsius to Fahrenheit  
✔️ Fahrenheit to Celsius  
✔️ Celsius to Kelvin  
✔️ Kelvin to Celsius  
✔️ Runs in a loop until the user exits  
✔️ Simple and beginner-friendly 

---

 Formulas Used

- **C → F** = (C × 9/5) + 32  
- **F → C** = (F − 32) × 5/9  
- **C → K** = C + 273  
- **K → C** = K − 273
