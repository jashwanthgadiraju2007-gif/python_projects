#  Dictionary API Project (Python)

A simple and interactive **Dictionary Application** built using Python.  
The user can search for the **meaning, two synonyms, and an example sentence** for any English word using a **free online Dictionary API**.

---

##  This Project Demonstrates

- ✅ API integration using `requests`  
- ✅ JSON data extraction  
- ✅ Conditional logic  
- ✅ Loops and user interaction  

---

##  Features

- Search meaning of any English word  
- Displays **two synonyms**  
- Shows **example sentence**  
- Handles invalid words safely  
- Runs in terminal / command prompt  
- Clean and beginner-friendly code  

---

 How the Program Works

1. User enters a word  
2. Program sends request to the Dictionary API  
3. API returns:
   - Meaning  
   - Synonyms  
   - Example  
4. Program displays the result in clean format  
5. User can continue searching or type **exit** to stop  

---

##  Sample Output
Enter a word: happy

 WORD: happy
 Meaning: Feeling or showing pleasure or contentment
 Synonym 1: joyful
 Synonym 2: glad
 Example: She felt happy after winning.
